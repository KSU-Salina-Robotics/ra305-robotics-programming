# RA305 - Robotics Programming

This repository supports the instructional activities, labs, and programming projects for RA305 – Robotics Programming within the Robotics & Automation Engineering Technology (RAET) program.

RA305 focuses on the software development practices required for robot systems, including real-time input/output (I/O), concurrency, control, teleoperation, autonomous algorithms, and network communication between a robots and controllers.

The materials in this repository are used to:

- Provide example implementations and reference code

- Support lab exercises and applied projects

- Reinforce lecture concepts with hands-on programming

- Enable testing in both simulation and physical robotic platforms

This repository serves as a central resource for course-related development and system integration activities.
